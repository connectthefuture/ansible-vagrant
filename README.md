# ansible-vagrant
This Ansible role can be used to install and maintain a vagrant virtual environment. (Tested on MacOS Sierra and Arch Kernel 4.10.8-1.)

## Requirements

* MacOS Sierra
* Arch Linux Kernel 4.10.8-1 or later.
* Ansible 2.0 or later.
